DASHBOARD - TABLEAU DE BORD

Généralités

1- Les séries statistiques:

a) historique : 24 mois

b) Fréquence : 1 mois

c) Ventilation par rôle (étudiant, professeur, bibliothécaire, IP, …)

 

2- Les KR

a) Historique : 12 mois

b) 1 KR par mois Vs. 1 KR de référence calculé sur l’ensemble d’une plateforme (ScholarVox, BiblioVox, …)

3- Le TOP présente les 50 premiers livres de la série statistique et du mois sélectionnés.

4- Le BOTTOM présente les 10 derniers livres de la série statistique et du mois sélectionnés.

5- En cliquant sur le titre #LIVRES, le TOP/BOTTOM des livres consultés agrégé sur les 24 derniers mois s’affiche.

6- En cliquant sur le titre #PAGES, le TOP/BOTTOM des pages consultées agrégé sur les 24 derniers mois s’affiche.

7- En cliquant sur une couverture de livre, le graphique de droite présente les statistiques pour le livre sélectionné sur les 24 derniers mois.

8- En cliquant sur le titre #LIVRE (graphique de droite), la fiche du livre est ouverte dans un nouvel onglet.

Séries statistiques #LIVRES

Livres consultés

Le nombre de consultations pour un livre correspond au nombre de sessions distinctes par livre. Le TOP/BOTTOM associé correspond aux livres qui se trouvent le plus/moins souvent dans les sessions d’utilisateurs.

Livres uniques consultés

Nombre de livres distincts consultés. Cette série ne donne pas accès au TOP/BOTTOM parce que les livres ne sont pas triés. Cette statistique permet de se rendre compte de la part du catalogue consultée.

Livres ajoutés

Nombre de fois qu’un livre est ajouté dans une étagère, une sélection ou un dossier. Le TOP/BOTTOM associé correspond aux livres les plus/moins enregistrés dans des étagères.

Séries statistiques #PAGES

Pages consultées (action code 10)

Nombre de pages consultées (selon la définition prévue par le code 10).

Pages imprimées (action code 9)

Nombre de pages imprimées (selon la définition prévue par le code 9)

Séries statistiques #COMPTES

Comptes actifs (action code 6 - compte oblige (hors IP))

Le nombre de comptes actifs correspond au nombre distinct de peopleID

Connexions (action code 6 - compte oblige (hors IP))

Le nombre de connexions correspond au nombre non distinct de peopleID. Pour information, il est difficile d’utiliser la notion de sessionID pour calculer le nombre de connexions par IP car on ne sait pas, par exemple, si une session reste active en connexion IP sur un ordinateur partagé pendant un temps indéterminé (1 jour ou plus, ou moins).

Séries statistiques #LIVRE

Livres consultés

Nombre de sessions dans lesquelles le livre est ouvert au moins une fois

Livre ajouté

Nombre de fois que le livre a été ajouté à une étagère, dossier, sélection.

Pages vues

Nombre de pages consultées pour ce livre (cf. Code 10).

Pages imprimées

Nombre de pages imprimées pour ce livre (cf. Code 9).

KEY RATIOS

KeyRatio Pages consultées par Compte Actif

Moyenne sur les 12 derniers mois glissants du nombre de pages consultées par nombre distinct de comptes actifs. Les pages consultées sont seulement celles qui sont consultées à partir d’un compte identifié (à l’inverse d’une page consultée par IP). Un compte actif est un compte qui s’est connecté au moins une fois pendant le mois correspondant.

KeyRatio Pages consultées par Session IP

Moyenne sur les 12 derniers mois glissants du nombre de pages consultées (par IP) par nombre total de sessions (sur le mois). Le nombre total de sessions IP est la statistique qui se rapproche le plus du nombre distinct de comptes actifs (tests également effectués sur le nombre de comptes contractuels ou une portion mais les KR étaient proches de zéro - donc insignifiant).

KeyRatio Livres consultés par Compte Actif

Moyenne sur les 12 derniers mois glissants du nombre de livres distincts consultés par compte actif. Les livres consultés sont seulement ceux qui sont consultés à partir d’un compte identifié (à l’inverse d’un livre consulté par IP). Un compte actif est un compte qui s’est connecté au moins une fois pendant le mois correspondant.

KeyRatio Livres consultés par Session IP

Moyenne sur les 12 derniers mois glissants du nombre de livres distincts consultés par IP. Les livres consultés sont seulement ceux qui sont consultés par IP (à l’inverse d’un livre consulté à partir d’un compte identifié). Le nombre total de sessions IP est la statistique qui se rapproche le plus du nombre distinct de comptes actifs (tests également effectués sur le nombre de comptes contractuels ou une portion mais les KR étaient proches de zéro - donc insignifiant).

KeyRatio de référence plateforme des Pages consultées par Compte Actif

Médiane des KeyRatio des Pages consultées par Comptes Actifs (cf. définition ci-dessus) des institutions contenues dans la plateforme concernée.

KeyRatio de référence plateforme des Pages consultées par Session IP

Médiane des KeyRatio des Pages consultées par Session IP (cf. définition ci-dessus) des institutions contenues dans la plateforme concernée.

KeyRatio de référence plateforme des Livres consultés par Compte Actif

Médiane des KeyRatio des Livres consultés par Compte Actif (cf. définition ci-dessus) des institutions contenues dans la plateforme concernée.

KeyRatio de référence plateforme des Livres consultés par Session IP

Médiane des KeyRatio des Livres consultés par Session IP (cf. définition ci-dessus) des institutions contenues dans la plateforme concernée.

 

KeyRatio proportion des connexions par Compte Actif

Pourcentage de pages consultées à partir d'un compte identifié = (1-KRproportion des connexions par IP)

 

KeyRatio proportion des connexions par Session IP

Pourcentage de pages consultées par Session IP = (1-KRproportion des connexions par compte actif)

 

Moyenne mensuelle des Pages consultées sur les 12 derniers mois

Moyenne sur 12 mois glissants du nombre de pages consultées (comptes actifs et sessionIP confondus)

Moyenne mensuelle des Livres consultés sur les 12 derniers mois

Le nombre de livres consultés sur un mois est la somme des nombres de livres distincts consultés par sessionIP et des livres distincts consultés par compte actif. Ensuite il s’agit d’une moyenne sur 12 mois glissants.
